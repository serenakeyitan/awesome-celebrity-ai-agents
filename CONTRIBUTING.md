## Adding to this list

Please ensure your pull request adheres to the following guidelines:

- Search previous suggestions before making a new one, as yours may be a duplicate.
- Make sure the celebrity/public figure has actually launched a personal AI agent or digital twin.
- Make an individual pull request for each suggestion.
- Use the following format: `| Person Name | AI Agent Name | Platform | Type | Description |`
- Link to the platform or official announcement where applicable.
- New categories or improvements to the existing categorization are welcome.
- Check your spelling and grammar.
- Make sure your text editor is set to remove trailing whitespace.
- The pull request and commit should have a useful title.
- The body of your commit message should contain a link to the source/announcement.

## Updating your Pull Request

Sometimes, maintainers will ask you to edit your Pull Request before it is included. This is normally due to spelling errors or because your PR didn't match the guidelines.

## Requirements

For a celebrity AI agent/clone to be included in this list, it must meet the following criteria:

1. **Officially Released**: The AI agent must be publicly available or officially announced by the celebrity or their team.
2. **Verifiable**: There must be credible sources (news articles, official announcements, platform listings) confirming the AI agent's existence.
3. **Actually AI**: The entry must be an actual AI agent, digital twin, voice clone, or AI avatar - not just a social media account or website.
4. **Celebrity/Public Figure**: The person must be a recognized public figure, celebrity, tech leader, or influential creator.

## Quality Standards

To be included on this list, entries should:

- Have a clear and accurate description
- Include the correct platform information
- Specify the type of AI (Digital Twin, Voice Clone, AI Avatar, Conversational AI, etc.)
- Be notable or newsworthy (significant media coverage, innovative approach, or substantial user base)
- Be actively available or have been officially announced (defunct projects should be marked as such)

## Categories

Entries are organized into the following categories:

- **Tech Leaders**: Technology entrepreneurs, CEOs, and industry leaders
- **Wellness**: Health, fitness, meditation, and personal development experts
- **Authors**: Writers, thought leaders, and content creators
- **Entertainment**: Actors, TV personalities, and entertainers
- **Music**: Musicians, singers, and music artists
- **Business**: Business coaches, investors, and entrepreneurs
- **Historical**: Deceased or historical figures recreated as AI

If you believe a new category is needed, please propose it in your pull request with at least 2-3 entries.

## Style Guide

- Use title case for person names
- AI agent names should match official branding
- Descriptions should be concise (1-2 sentences max)
- Types should be one of: Digital Twin, Voice Clone, AI Avatar, Conversational AI, De-aging AI, AI Companion
- Maintain alphabetical order within each category
- Use consistent table formatting

Thank you for your suggestions!
